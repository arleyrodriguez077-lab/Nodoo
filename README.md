# Red Social: Comunidad Mental

Una plataforma inclusiva y abierta para el apoyo mutuo sobre temas de psicosis.

## Características
- Autenticación segura con Firebase.
- Perfiles de usuario personalizados.
- Blog comunitario sin juicios (sin likes ni amigos).
- Entorno de apoyo inclusivo.

## Tecnologías
- HTML5, CSS3, JavaScript.
- Firebase Authentication y Firestore.

## Instalación
1. Clona este repositorio.
2. Crea un proyecto en [Firebase Console](https://console.firebase.google.com/).
3. Reemplaza los datos en `app.js` con tu configuración de Firebase.
4. Despliega usando **GitHub Pages** en la configuración de este repositorio.

## Aviso importante
Esta comunidad es un entorno de apoyo y **no reemplaza la atención médica profesional**. Si estás en crisis, busca ayuda en tu centro de salud local.
